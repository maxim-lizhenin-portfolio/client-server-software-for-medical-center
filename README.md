# Название

client-server-software-for-medical-center

***


# Описание

Проект представляет собой клиент-серверное приложение для медицинского центра. 
Клиент-приложение реализовано на Godot Engine. Сервер реализован на Python с использованием FastAPI. Также реализована база данных на PostgreSQL, с которой взаимодействует сервер.
Клиент (работник больницы или посетитель) в зависимости от своей роли может получать или редактировать данные в базе данных больницы.
